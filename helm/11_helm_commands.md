<table>
<tr>
<th>Command</th>
<th>Description</th>

<tr>
<td><code>helm upgrade</code></td>
<td>Upgrade release to a new version of a chart</td>
</tr>

<tr>
<td><code>helm rollback</code></td>
<td>
<ul>
<li>Rollback to a previous version (revision) </li>

</ul>
</td>
